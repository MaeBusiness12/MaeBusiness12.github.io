Block Blast

Overview

Block Blast is a 2D game where you stach blocks on a 8x8 grid.

How to Play

The game is based on and reminisant of the classic game Tetris. When Block Black begins, the player is given an 8x8 blank grid where 3 blocks appear. In order to proceed in the game, the player must:

  1. Drag and drop blocks.
  2. Create vertical and horizontal lines without gaps.
  3. Destroy existing blocks by creating the lines.

Adventure Mode

In Block Blast Adventure Mode, each new level presents a new challenge. Pre placed blocks allow the player to begin with a more strategic approach.
